Portfólio

Projeto em Equipe - Pharmacy Central System API

O Pharmacy Central System API é uma solução para gerenciamento de usuários, medicamentos e vendas. Desenvolvida em JavaScript com Express.js, a API utiliza PostgreSQL e Sequelize para o gerenciamento de dados. Todos os endpoints são protegidos por token JWT, com exceção do cadastro e login de usuários. Você pode visualizar o projeto no Vercel através do link:
