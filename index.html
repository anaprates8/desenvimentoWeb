Portfólio

Projeto em Equipe - Pharmacy Central System API

O Pharmacy Central System API é uma solução para gerenciamento de usuários, medicamentos e vendas.
Desenvolvida em JavaScript com Express.js, a API utiliza PostgreSQL e Sequelize para o gerenciamento de dados.
Todos os endpoints são protegidos por token JWT, com exceção do cadastro e login de usuários.

Você pode visualizar o projeto no Vercel através do link: Pharmacy Central System API no Vercel.

Para mais detalhes sobre os endpoints, acesse a documentação em https://pcs-api-56ex.onrender.com/api-docs ou execute localmente em http://localhost:3333/api-docs/.

Veja o código no GitHub: Pharmacy Central System API
